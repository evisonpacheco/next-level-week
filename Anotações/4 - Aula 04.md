## Arrow Function
Forma contraída de uma função.

Quando apenas uma ação vai ser executada na arrow function não é necessário {}

* Sem argumento:
```js
() => {

}
```

* Um argumento:
```js
argumento => {

}
```

* Quando a função recebe mais de um argumento:

```js
(argumento, argumento2) => {

}
```
